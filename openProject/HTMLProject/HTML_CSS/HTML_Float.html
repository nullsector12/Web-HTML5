치항옳

헬스클럽 전문 트레이너인 HealthChang씨는 군 전역 후 생활체육과를 졸업하고 20대 초반부터 쭉 트레이너를 해온 베테랑 트레이너이다. 현재 소속된 헬스클럽에서 5년간 다양한 담당회원을 만나 관리하면서 그에 대한 입소문이 퍼져 현재는 월별 관리 회원이 무려 20명 이상이 된다.

그러나 HealthChang씨는 담당회원이 20명이 넘어가면서 개별 회원에 대한 상세한 케어나 일정관리 등에서 문제가 발생하였다.

따라서 효율적으로 회원들을 관리 할 수 있는 기능을 담은 웹&앱 서비스를 원했다. 메인 디바이스는 운동을 하거나 이동 중에도 항상 확인 할 수 있는 앱서비스를 기반으로 웹과도 연동되도록 하는 서비스이다. HealthChang씨는 해당 서비스에 회원과의 1:1채팅과 일정관리 기능을 기본으로 회원들의 식단관리, 양방향 일정공유 기능을 원했다.

새로운 서비스 기능을 잘 활용하지 못할 이용자들을 위한 쉽고 자세한 설명문을 앱&웹화면 어디서든 확인 할 수 있도록 하는 기능을 통해 회원들이 겪을 어려움을 해결했다.
